#+title: Notes

* Opcodes
| Opcode       | Type            |
|--------------+-----------------|
| 0000011  (3) | Loads           |
| 0010011 (19) | I-type          |
| 0010111 (23) | auipc           |
| 0100011 (35) | Stores          |
| 0110011 (51) | R-type (or) lui |
| 1100011 (99) | Branches        |
| 1100111(103) | jalr            |
| 1101111(111) | jal             |

| aluOp | Operation         |
